# LemonDoodle-Vij

LemonDoodle for APCSA





Notes on DoodlePad:

* Use the specific compiling and executing commands from doodlepad
* The library gives access to numerous animation, shape-building, and graphics tools
* Animations can be made by using a sprite image and looping through the image
* doodlepad.jar must be in the same folder as the file that is being run
* import doodlepad.\* (import to use the DoodlePad library)
* If using and image, the image should be in the same folder as the running file
